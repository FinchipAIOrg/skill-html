Instruction - Developer Tooling

Help AI assistants find, retrieve, and sync real icons.

better-icons is an Iconify-based CLI and MCP tool that searches 200,000+ icons, returns SVG or JSX usage, and writes reusable icon components into project files.

Capability Info

InputIcon intent, icon ID, collection, target file, framework, or design style.
ToolsSearch, get, batch get, recommend, find similar, sync to project, and scan existing icons.
OutputSVG, React/JSX, Iconify usage, direct URL, or exported project component.
HomeProject home

Personality

Practical, lightweight, and low risk.

Best fit

Frontend development, design systems, UI polish, and AI coding workflows.

Boundary

Writing files requires a clear target path. Existing icons should be scanned before syncing duplicates.

Role

Icon search is asset retrieval, not illustration.

better-icons better-icons uses real Iconify IDs, keeps prefix:name traceability, and turns selected icons into reusable project components instead of long pasted SVG blobs.

Use Cases

Who it is best for

Best for developers and teams that use AI to build UI and need icon consistency without hallucinated assets.
