Use it where evidence changes the decision.

Optimized for analytical business communication, not generic slideshow generation.

Strong fit

  • Executive and board updates
  • Quarterly business reviews
  • Strategy and market analyses
  • Options comparisons and recommendations
  • Client findings with auditable sources

Not the right tool

  • Photo-heavy event slideshows
  • Purely artistic keynote design
  • Unverified viral-statistic compilations
  • A substitute for regulated professional review
  • Native editable PPTX as the only output

Give it sources, context, and a real decision.

Dependencies

HTML generation and validation require only Python 3. PDF export additionally requires Chrome, Chromium, Edge, or Brave. The optional visual regression check uses Playwright.

Boundaries

Source quality remains a human judgment. The Skill exposes unsupported reasoning but cannot guarantee that a cited source is true, current, or suitable for a regulated decision.
