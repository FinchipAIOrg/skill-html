Evaluation Standard Integration

Gate, score, and operate checks

Standard areaThis benchmark now checksReject or watch signals
GateThe skill must only proceed when the target is a real listed equity or ETF and enough public data is available. It must disclose missing data before generating a final HTML report.Hallucinated filings, unsupported private data, or direct trading instructions fail the gate.
Weighted scoreScore now combines stage completion, data freshness labels, valuation-model disagreement, panel debate quality, trap-risk integration, cost per completed report, and explicit safety framing.A long report is not enough; incomplete stage evidence or missing risk blocks should lower the score.
Three-run stabilityRun the same ticker three times. The main business facts, risk flags, valuation range direction, and missing-data notes should stay stable even if narrative phrasing changes.Large swings in core conclusion without new data indicate unstable research behavior.
Agent outputThe benchmark expects reusable HTML plus structured scores, data-source notes, panel consensus, valuation assumptions, and unresolved checks for another agent to continue.Plain prose that cannot feed a portfolio-review or audit workflow is insufficient.
Regression setMaintain A-share, HK-share, US-share, ETF, and ambiguous-name cases as regression inputs before commercial release.Any run that hides unavailable data or presents a price target as advice is a blocker.
