Four gates. Different kinds of proof.

A deterministic pass is necessary for delivery, but it is not a substitute for an editor's judgment.

GATE 01

Structure

Valid Skill metadata, JSON shape, stable IDs, required fields, relative file paths, and parseable artifacts.

GATE 02

Traceability

Known atom references, exact quotes, caveat companions, forbidden phrases, review flags, and character limits.

GATE 03

Delivery

Manifest-to-file parity, expected file set, proposed schedule references, and a machine-generated QA report.

GATE 04 · HUMAN

Editorial release

Truth, rights, voice, platform fit, current policy, confidentiality, regulated claims, and final approval.

Known limits stay visible.

The benchmark is strongest where deterministic evidence exists and explicit where it does not.

No performance guarantee

The tests do not predict reach, engagement, leads, virality, ranking, or conversion.

No truth oracle

A traceable source can still be wrong, outdated, confidential, or unsuitable for publication.

No live platform audit

Platform limits and features change; current official rules must be checked before publishing.

No visual production

Carousel and short-video briefs do not prove that finished images or videos exist.

No publishing test

The Skill deliberately stops at proposed scheduling. External posting needs separate authorization.

UI trace unavailable

Local Codex and terminal apps blocked screenshot capture; evidence is therefore file-backed and explicitly labeled.
